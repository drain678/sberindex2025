# Проект "Арктические переселения" 

## Сравнительный анализ Катунинского и Калевальского поселений
Проект сравнивает экономику, жильё, образование, здравоохранение и инфраструктуру двух поселений на уровне нижних муниципальных образований с использованием данных ПОРА и СберИндекса. Анализ выполнен в Jupyter Notebook с визуализациями и выводами по ключевым показателям качества жизни.
